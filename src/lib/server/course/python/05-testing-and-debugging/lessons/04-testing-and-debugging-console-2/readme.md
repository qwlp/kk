# Fixing a Wrong Variable

A small naming mistake can break working logic.

```python
hearts = 3
print(hearts)
```

Here is what the example does:

- `hearts` stores the number `3`.
- `print(hearts)` displays the value stored in `hearts`.

The assignment changes the specific values, but the basic pattern is the same.

## Assignment

Complete the starter code so the program does the following:
