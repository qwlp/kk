# Updating a Dictionary

Dictionaries are mutable, so you can replace a value for a key.

```python
bag = {"arrows": 4}
bag["arrows"] = 7
print(bag["arrows"])
```

Here is what the example does:

- `bag` creates a dictionary and stores it in `bag`.
- `bag['arrows']` stores the number `7`.
- `print(bag['arrows'])` displays the result of `bag['arrows']`.

The assignment changes the specific values, but the basic pattern is the same.
