# Using Multiple Dictionary Values

One dictionary can hold several related values about the same thing.

```python
pet = {"type": "fox", "age": 4}
print(pet["type"])
print(pet["age"])
```

Here is what the example does:

- `pet` creates a dictionary and stores it in `pet`.
- `print(pet['type'])` displays the result of `pet['type']`.
- `print(pet['age'])` displays the result of `pet['age']`.

The assignment changes the specific values, but the basic pattern is the same.
