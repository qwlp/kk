# Practice: Inventory Summary

Combining dictionaries and arithmetic starts to feel like real data modeling.

```python
points = {"wins": 3, "bonus": 2}
print(points["wins"] + points["bonus"] )
```

Here is what the example does:

- `points` creates a dictionary and stores it in `points`.
- `print(points['wins'] + points['bonus'])` displays the result of `points['wins'] + points['bonus']`.

The assignment changes the specific values, but the basic pattern is the same.
