# Adding to a List

Lists can change over time.

```python
pets = ["cat", "dog"]
pets.append("owl")
print(len(pets))
```

Here is what the example does:

- `pets` creates a list and stores it in `pets`.
- `pets.append('owl')` calls `pets.append` with the given input values.
- `print(len(pets))` runs `len(pets)` and displays the returned value.

The assignment changes the specific values, but the basic pattern is the same.
