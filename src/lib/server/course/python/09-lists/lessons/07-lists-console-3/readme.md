# Looping Over a List

Lists and loops fit together naturally.

```python
sounds = ["drip", "clang", "boom"]
for sound in sounds:
    print(sound)
```

Here is what the example does:

- `sounds` creates a list and stores it in `sounds`.
- The `for` loop takes each item from `sounds` one at a time and puts it into `sound`.
- `print(sound)` displays the value stored in `sound`.

The assignment uses different values, but you will use the same loop pattern.
