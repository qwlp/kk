# Looping Over Words

Loops work with strings just as well as numbers.

```python
words = ["sun", "moon"]
for word in words:
    print(word)
```

Here is what the example does:

- `words` creates a list and stores it in `words`.
- The `for` loop takes each item from `words` one at a time and puts it into `word`.
- `print(word)` displays the value stored in `word`.

The assignment uses different values, but you will use the same loop pattern.
