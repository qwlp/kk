# Defining and Calling a Function

Defining a function gives a reusable block of code a name. After you define it once, you can call it whenever you want that code to run.

```python
def cheer():
    print("Go!")

cheer()
```

Here is what the example does:

- `def cheer()` defines a function named `cheer`.
- Inside the function, `print('Go!')` displays the text "Go!".
- `cheer()` calls `cheer` so its code runs.
