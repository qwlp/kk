# Reading Code Top to Bottom

Python runs a file from top to bottom, so line order matters.

```python
print("Open")
print("Run")
print("Close")
```

Here is what the example does:

- `print('Open')` displays the text "Open".
- `print('Run')` displays the text "Run".
- `print('Close')` displays the text "Close".

The assignment changes the specific values, but the basic pattern is the same.
