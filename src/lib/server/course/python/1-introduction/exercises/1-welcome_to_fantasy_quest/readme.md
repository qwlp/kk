# Welcome to Fantasy Quest

You're about to start building a tiny text-based RPG in Python.

For your first lesson, all you need to do is print the correct welcome message for the player.

## Assignment

Fix the string in the starter code so the program prints:

`Welcome to Fantasy Quest!`

## Tips

- Python uses `print(...)` to write text to the terminal.
- Pay close attention to capitalization and spelling.
- The grader checks the exact output.
