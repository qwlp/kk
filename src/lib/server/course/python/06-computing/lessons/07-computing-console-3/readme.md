# Converting Kilobytes to Bytes

A kilobyte is commonly treated as 1024 bytes in beginner computing lessons.

```python
megabytes = 2
kilobytes_per_megabyte = 1024
total_kilobytes = megabytes * kilobytes_per_megabyte
print(total_kilobytes)
```

Here is what the example does:

- `megabytes` stores the number `2`.
- `kilobytes_per_megabyte` stores the number `1024`.
- `total_kilobytes` calculates `megabytes * kilobytes_per_megabyte` and stores the result.
- `print(total_kilobytes)` displays the value stored in `total_kilobytes`.

The assignment changes the specific values, but the basic pattern is the same.
