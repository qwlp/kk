# Using `if` With a Boolean

`if` runs code only when its condition is true.

```python
is_logged_in = True
if is_logged_in:
    print("Welcome")
```

Here is what the example does:

- `is_logged_in` stores the Boolean value `True`.
- The `if` statement checks whether `is_logged_in` is `True`.
- `print('Welcome')` displays the text "Welcome".

The assignment changes the specific values, but the basic pattern is the same.
