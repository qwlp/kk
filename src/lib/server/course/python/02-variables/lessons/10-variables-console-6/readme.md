# Calculating a Remaining Value

Variables are useful for modeling changing state.

```python
total_arrows = 12
used_arrows = 5
arrows_left = total_arrows - used_arrows
print(arrows_left)
```

Here is what the example does:

- `total_arrows` stores the number `12`.
- `used_arrows` stores the number `5`.
- `arrows_left` calculates `total_arrows - used_arrows` and stores the result.
- `print(arrows_left)` displays the value stored in `arrows_left`.

The assignment changes the specific values, but the basic pattern is the same.
