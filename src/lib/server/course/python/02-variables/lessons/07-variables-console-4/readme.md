# Printing Labels and Variables

A label makes output easier to read.

```python
quests = 3
reward_per_quest = 5
points = quests * reward_per_quest
print("Points:")
print(points)
```

Here is what the example does:

- `quests` stores the number `3`.
- `reward_per_quest` stores the number `5`.
- `points` calculates `quests * reward_per_quest` and stores the result.
- `print('Points:')` displays the text "Points:".
- `print(points)` displays the value stored in `points`.

The assignment changes the specific values, but the basic pattern is the same.
