# Updating a Variable

Variables can change over time.

```python
arrows = 5
arrows = arrows + 2
print(arrows)
```

Here is what the example does:

- `arrows` stores the number `5`.
- `arrows` calculates `arrows + 2` and stores the result.
- `print(arrows)` displays the value stored in `arrows`.

The assignment changes the specific values, but the basic pattern is the same.
