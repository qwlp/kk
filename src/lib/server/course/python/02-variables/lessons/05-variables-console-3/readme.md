# Using Variables in Calculations

Variables make calculations easier to read.

```python
base_score = 6
bonus_score = 4
total_score = base_score + bonus_score
print(total_score)
```

Here is what the example does:

- `base_score` stores the number `6`.
- `bonus_score` stores the number `4`.
- `total_score` calculates `base_score + bonus_score` and stores the result.
- `print(total_score)` displays the value stored in `total_score`.

The assignment changes the specific values, but the basic pattern is the same.
