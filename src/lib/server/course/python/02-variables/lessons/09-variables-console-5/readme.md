# Reusing a Variable

Store a value once and use it more than once.

```python
pet = "dragon"
print(pet)
print(pet)
```

Here is what the example does:

- `pet` stores the text "dragon".
- `print(pet)` displays the value stored in `pet`.

The assignment changes the specific values, but the basic pattern is the same.
